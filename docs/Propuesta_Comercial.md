# Propuesta Comercial: Plataforma "TutoresOn-Line"

## 1. Resumen Ejecutivo
Desarrollo llave en mano de **"TutoresOn-Line"**, una plataforma de tutorías diseñada para conectar estudiantes con tutores calificados. Incluye desarrollo web responsivo (accesible desde móviles y web), sistema de reservas, integraciones de video y analítica apoyada por Inteligencia Artificial.

## 2. Alcance del Proyecto
- Módulo de inicio de sesión, registro seguro y perfiles de usuario.
- Búsqueda avanzada de tutores (por materia, nivel educativo, disponibilidad y **cercanía geográfica**).
- Sistema de Reservas transaccional con control de solapamiento.
- Gestión de Disponibilidad dinámica (horarios).
- Videoconferencias embebidas y/o generación automatizada de links.
- Módulo de reputación (Sistema de reseñas 1 a 5 estrellas).
- Integración GenAI para asistencia académica previa y *matchmaking*.
- Notificaciones vía SMS y Email de las actividades de reserva.

## 4. Estimación de Tiempo (Cronograma Ágil)
Para cumplir con las expectativas de un producto de grado corporativo, se estructuran **5 Sprints principales**, calculados semanalmente según progreso empírico:

- **Sprint 1:** Análisis de requerimientos, diseño UI/UX (pantallas base), Arquitectura de BD y Arquitectura Hexagonal Backend.
- **Sprint 2:** Inicialización, Auth, y módulos de perfil (Alumno y Tutor).
- **Sprint 3:** Motor de búsquedas avanzadas (por geografía e índices), módulo de disponibilidad y solicitudes manuales.
- **Sprint 4:** Matchmaking en vivo, integraciones AI Generativas y APIs de terceros (Notificaciones, WebRTC/Video).
- **Sprint 5:** Estabilización, Refinamientos (UI premium micro-animaciones), QA y producción de Manuales.
